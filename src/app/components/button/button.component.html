Buttons

The hover-button component provides customizable buttons with ripple effects, icons, size options, and color variants. These are ideal for actions in your UI.

Available Inputs

Attribute Description Possible Values
color Button color style primary, secondary, complete
size Defines button size sm, md, lg
disabled Disables the button when set to 'disabled' disabled | ''
icon Optional icon displayed before content Material icon name

Button Color Variants

These buttons showcase the available color variants (primary, secondary, and complete), which help visually distinguish different actions or importance levels.

Primary Secondary Complete
copy 

    <hover-button color='primary'>Primary</hover-button>
    <hover-button color='secondary'>Secondary</hover-button>
    <hover-button color='complete'>Complete</hover-button>

Button Sizes

The size input allows you to choose between small (sm), medium (md), and large (lg) button styles to fit different contexts in your layout.

Small Medium Large
copy 

      <hover-button color='primary' size='sm'>Small</hover-button>
      <hover-button color='secondary' size='md'>Medium</hover-button>
      <hover-button color='complete' size='lg'>Large</hover-button>

Disabled Buttons

Set disabled="disabled" to visually and functionally disable the button. Useful for inactive states or waiting actions.

Primary Secondary Complete
copy 

    <hover-button color='primary' disabled='disabled'>Primary</hover-button>
    <hover-button color='secondary' disabled='disabled'>Secondary</hover-button>
    <hover-button color='complete' disabled='disabled'>Complete</hover-button>

Buttons with Icons

You can add icons to the button using the icon input. Icons are placed before the text and use the Angular Material Icons set.

Save Home Lock
copy 

      <hover-button color='primary' icon='save' size='sm'>Save</hover-button>
      <hover-button color='secondary' icon='home' size='md'>Home</hover-button>
      <hover-button color='complete' icon='lock' size='lg'>Lock</hover-button>
