Alerts

The hover-alert component allows you to display various types of alerts to users, such as information, success, error, and warning. These alerts come with customizable features like manual close, automatic close, progress bars, optional titles, and more. Below are examples of each type of alert with descriptions of their available attributes.

Available Inputs

Attribute Description Possible Values
type Defines the type of the alert info, success, error, warning
message The message text to display in the alert Any string
title Optional title for the alert (not used for all types) Any string
closable Indicates whether the alert can be manually closed by the user true, false
autoClose Automatically closes the alert after a specified duration true, false
duration The time (in milliseconds) before the alert auto closes Any positive integer (e.g., 5000 for 5 seconds)
showProgressBar Displays a progress bar while the alert is visible true, false

Info

This alert is typically used to display informational messages. It has the option of being closed manually by the user. It can be customized with a message and can be made closable using the closable attribute.

copy 
      
        <hover-alert
        type='info'
        message='This is an info alert.'
        [closable]='true'
        ></hover-alert>

Success

Success alerts are useful for displaying positive feedback or confirmation. They usually include a title, like "Success", and a message with additional information. The closable option allows the user to close the alert manually.

copy 
      
        <hover-alert
        type='success'
        title='Success'
        message='The data was saved successfully.'
        [closable]='true'
        ></hover-alert>

Error

Error alerts are used to notify users of problems or issues that require attention. You can customize this alert with a title, message, and set properties like showProgressBar and autoClose. Additionally, you can control how long the alert is visible using the duration attribute.

copy 
      
        <hover-alert
        type='error'
        title='Error'
        message='The operation could not be completed.'
        [showProgressBar]='true'
        [autoClose]='true'
        [duration]='5000'
        ></hover-alert>

Warning

Warning alerts are used to highlight potential issues that require attention but are not critical. You can set a title, provide a message, and control whether the alert is closable using the closable attribute.

copy 
      
        <hover-alert
        type='warning'
        title='Warning'
        message='Please check the input data.'
        [closable]='false'
        ></hover-alert>
