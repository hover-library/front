Hover Spinner

The hover-spinner component provides animated loading indicators to improve user experience while data is loading or a process is running. It supports multiple styles, sizes, and color themes to fit various design needs.

Spinners are ideal for asynchronous operations, form submissions, or content placeholders. They are lightweight, customizable, and easy to use.

Types

The type input lets you choose from four unique spinner styles: pulse, wave, cube, and neo. Each style is animated with its own pattern and feel.

copy 

<hover-spinner type='pulse' color='primary'></hover-spinner>
<hover-spinner type='wave' color='tech'></hover-spinner>
<hover-spinner type='cube' color='accent'></hover-spinner>
<hover-spinner type='neo' color='neon'></hover-spinner>

Sizes

You can define the spinner's size using the size input. Available options are sm, md (default), and lg. Use different sizes to fit spinners into buttons, cards, or full-page loaders.

copy 

<hover-spinner size='sm'></hover-spinner>
<hover-spinner size='md'></hover-spinner>
<hover-spinner size='lg'></hover-spinner>

Colors

Spinners can be styled with theme colors using the color input. The available values are primary, tech, accent, and neon. These follow the design system of your application.

copy 

<hover-spinner color='primary'></hover-spinner>
<hover-spinner color='tech'></hover-spinner>
<hover-spinner color='accent'></hover-spinner>
<hover-spinner color='neon'></hover-spinner>
