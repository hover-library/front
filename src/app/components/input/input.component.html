Inputs

The hover-input component provides customizable input fields with support for icons, validation, and error messages. It is ideal for creating interactive forms with a clear interface.

Basic Inputs

In this example, it shows how to use the hover-input component to create basic fields like "Full Name" and "Password" with icons. These inputs are ideal for simple forms where text or password entries are required.

copy 

      <hover-input label='Full Name' icon='person'></hover-input>
      <hover-input label='Password' type='password' icon='lock'></hover-input>

Inputs with States

Here we show inputs that include different field types: email, phone, and password. You will also see a disabled input, which is achieved by using the [disabled]="true" property.

copy 

      <hover-input label='Email' type='email' icon='mail'></hover-input>
      <hover-input label='Phone' type='tel' icon='phone'></hover-input>
      <hover-input label='Disabled' disabled='true' icon='block'></hover-input>

Inputs with Reactive Validation

These inputs are connected to a reactive form with validation. We use controls such as email, password, and age, each with custom validation rules. It demonstrates how to handle errors and custom messages using the component.

Submit Form Clear

The reactive form allows for advanced validation, such as ensuring that the email field is correctly formatted and that the password has at least 8 characters. If the form contains errors, the "Submit Form" button is disabled.

copy 

      <form [formGroup]="form" (ngSubmit)="onSubmit()" class="form-reactive">
        <hover-input
          label="Corporate Email"
          type="email"
          formControlName="email"
          [control]="form.get('email')"
          icon="alternate_email"
          [customMessages]={{ "'{" }} required: 'This field is required', default: 'Enter a valid email address' {{ "}'" }}
        ></hover-input>

        <hover-input
          label="Password"
          type="password"
          formControlName="password"
          [control]="form.get('password')"
          icon="lock"
          [customMessages]={{ "'{" }} required: 'This field is required', minlength: 'Minimum 8 characters' {{ "}'" }}
        ></hover-input>

        <hover-input
          label="Age (optional)"
          type="text"
          formControlName="age"
          [control]="form.get('age')"
          helperText="Between 18 and 99"
          icon="elderly"
        ></hover-input>

        <div class="actions">
          <hover-button
            color="primary"
            type="submit"
            [disabled]="form.invalid ? 'disabled' : ''"
          >
            Submit Form
          </hover-button>

          <hover-button
            color="secondary"
            type="button"
            (click)="reset()"
            icon="delete"
          >
            Clear
          </hover-button>
        </div>
      </form>
